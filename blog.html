1) What is DOM? What is the purpose of DOM?

The ellaboration of DOM is Document Object Model. When web page is loaded on the browser there is create Document Object Model. The DOM is constructed as a tree. And its every element called Node.
Javascript accesses all the html page by using DOM. The javascript can change all the text or elements or attribute by the DOM. Javascript can add or remove any class or id. Javascript can happend event in any Node of DOM.

2) How will you select HTML elements using DOM?

There are many options to select HTML elements using DOM.
There are:-

  • document.getElementsByTagName()
  • document.getElementsByClassName()
  • document.getElementById()
  • document.querySelector()
  • document.querySelectorAll()

3) What is the Event Bubble?

In scripting language, event meaning when user click or reload the webpage, there is something happend. And it is called Event. When user click something on webpage then it's not only triggers the target element but also it triggers the parent element in the same nesting hierarchy, till it reaches outermost of DOM element. It is called Event Bubble.

4) What is call back function? Why will you use it?

A call back function is when a function is called as a arguement in another function. Javascript functions are executed by the sequence they are called, not the sequence they are defined. call back functions are work as a library. We can define a function anywhere of javascript file, and any time we called the function. No matter if it is in another function. That's why I use call back function. It makes the coding more easier.
